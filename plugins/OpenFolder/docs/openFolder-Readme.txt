Overview
--------
Adds a menuitem which opens the Miranda IM folder.
Supports MirInstaller, Uninstaller & TopToolbar Plugin.

Homepage & Forum
----------------
http://miranda-im.org/download/details.php?action=viewfile&id=1542

Information
-----------
'Open' the folder by clicking the item.
'Explore' the folder by holding CTRL & clicking the item.
